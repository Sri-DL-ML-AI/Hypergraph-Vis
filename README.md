# Hypergraph-Vis
Code repository for "Hypergraph Visualization via a Metric Space Viewpoint and Persistence"

## Requirements
HyperNetX (>=0.2.5)

## How to run

```bash
cd Hypergraph-Vis
python3 run.py
#Hit Ctrl+c to quit
```

You can view the page at http://0.0.0.0:6060/ (If possible, please use Chrome).
